# Revista Support

Looking for Support? Have a question? There are a few options:

* **Documentation** &mdash; Check out the [README.md](./README.md) for installation and usage instructions
* **Architecture Guide** &mdash; See [ARCHITECTURE.md](./ARCHITECTURE.md) for technical details
* **Content Providers** &mdash; Read [CONTENT_PROVIDERS.md](./CONTENT_PROVIDERS.md) for content source configuration
* **Contributing Guide** &mdash; See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to contribute
* **GitHub Issues** &mdash; [Report bugs or request features](https://github.com/chrisnmorrison/revista-astro-blog-magazine/issues)
* **GitHub Discussions** &mdash; [Ask questions and discuss ideas](https://github.com/chrisnmorrison/revista-astro-blog-magazine/discussions)

## Getting Help

1. **Check the documentation** - Most questions are answered in the README and other docs
2. **Search existing issues** - Your question might have already been answered
3. **Open a discussion** - For questions and general discussion
4. **Report bugs** - Use the Issues tab for bug reports and feature requests

On the GitHub Discussions board, there are helpful community members that can point you in the right direction.
